# تعديل تحديث منتصف الليل — 5.0.1

هذه الحزمة تجعل التحديث اليومي يعمل عند الساعة **12:00 منتصف الليل بتوقيت عمّان**.

## نقاط التحقق

1. لا يوجد Cron تلقائي عند 00:15 في GitHub Actions.
2. Workflow التحديث يعمل مرة واحدة عند 00:00 بتوقيت `Asia/Amman`.
3. التطبيق يحاول استخدام منبه Android دقيق عند سماح النظام.
4. يوجد WorkManager احتياطي في حال عدم منح صلاحية المنبه الدقيق.
5. يعاد ضبط الموعد بعد إعادة تشغيل الهاتف أو تغيير الوقت والمنطقة الزمنية.
6. عند غياب الإنترنت أو تأخر بيانات اليوم، تتم إعادة المحاولة تلقائيًا.

## ملاحظة Android

Android قد يقيّد العمل الدقيق في الخلفية على بعض الأجهزة. لذلك أضيف زر داخل إعدادات التطبيق لفتح صلاحية «المنبهات والتذكيرات»، مع بقاء الجدولة الاحتياطية وفحص فتح التطبيق فعالين.
