السبب: في الأيام التي لا توجد فيها قطعة اختيارية مثل طروبارية عيد أو آية مناولة خاصة، كان فاحص Python يستبدل علامة القالب بقيمة فارغة ثم يعتبر السطر الفارغ خطأ. تطبيق أندرويد كان يحذف هذه الأسطر أصلًا، لكن فاحص GitHub لم يكن يطبق التنظيف نفسه.

الإصلاح يجعل فاحص GitHub يطابق سلوك التطبيق:

- يحذف المقطع الاختياري إذا كان البديل فارغًا.
- يحذف علامات القالب التي لم تُحل.
- يحذف رسائل «غير متاح» القديمة.
- يحذف عنوان القسم إذا لم يبق تحته محتوى.
