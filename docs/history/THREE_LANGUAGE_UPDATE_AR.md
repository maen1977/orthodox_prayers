# نظام التحديث المستقل للغات

- تقويم بطريركية القدس يحدد تاريخ اليوم والمناسبة ومراجع القراءات.
- العربية: مطرانية عمان، ثم القدس، ثم أنطاكية.
- اليونانية: Apostoliki Diakonia، ثم GOARCH اليوناني، ثم القدس باليونانية.
- الإنجليزية: GOARCH، ثم القدس بالإنجليزية، ثم أنطاكية بالإنجليزية.
- OCN وPemptousia وAghioritiki Estia وRomfea مصادر تعليمية/مرجعية وليست سلطة تلقائية للنص الكتابي اليومي.
- لا ترجمة آلية ولا fallback بين اللغات. فشل لغة لا يمنع نشر اللغتين الأخريين.
