# تقرير Orthodox Prayers 4.0.0

- تم اعتماد مفتاح توقيع واحد جديد وربط المفتاح العام نفسه بالـPython وAndroid.
- أضيف فحص إلزامي داخل GitHub Actions يثبت أن الـSecret يطابق المفتاح العام المضمّن قبل التوقيع.
- أصبح مسار التحديث الرئيسي `scripts/update.py` ومسار التحقق `scripts/verify.py`.
- يبقى في GitHub Workflowان فقط: Build وUpdate.
- Update لا يعمل عند كل Commit؛ يعمل يدويًا أو الساعة 00:00 و00:15 بتوقيت عمّان.
- البيانات الجزئية الموثوقة تُنشر بدل حجب اليوم كاملًا، مع إبقاء النص غير الموثق فارغًا.
- لا توجد ترجمة تلقائية بين العربية والإنجليزية واليونانية.
- نجحت 63 وحدة اختبار وبوابة جودة المحتوى والتوقيع والـWorkflows.
- تعذر تشغيل Gradle محليًا فقط لأن بيئة الفحص لا تصل إلى services.gradle.org؛ GitHub Actions مهيأ لتشغيله.
