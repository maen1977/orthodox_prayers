# إصلاح تعطّل بيانات اليوم

المشكلة السابقة كانت أن نقص حقل رسمي واحد يحجب ملف اليوم كاملًا، فيبقى التطبيق على تاريخ قديم.

السلوك الجديد:
- ينشر تاريخ اليوم وحالة الصيام والعيد والخدمات والمراجع المتحققة يوميًا.
- ينشر النص الديني الكامل فقط عندما يكون من مصدر رسمي مطابق للغة.
- يترك الحقل غير المتحقق وحده فارغًا بدل حجب اليوم كله.
- يوقع ملف اليوم ويختبر التاريخ والتوقيع بعد النشر.
- لا يستخدم ترجمة بين العربية والإنجليزية واليونانية.

ملاحظة: اكتمال صلاة الصباح نفسها موضوع محتوى منفصل؛ المكتبة العربية الحالية تحتوي النصوص المثبتة فقط، ولن يتم اختراع بقية النص أو ترجمتها آليًا.
