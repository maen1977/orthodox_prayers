# ملاحظات إصدار 3.4.0

- فصل المحتوى إلى ثلاث مكتبات أصلية مستقلة: عربية ويونانية وإنجليزية.
- منع الترجمة الآلية ونسخ النص بين اللغات.
- إضافة سجل مصادر وموافقات وربط كل خدمة بمصدرها الرسمي.
- إضافة فحص اكتمال وبصمة SHA-256 لكل مكتبة وخدمة.
- إضافة أداة استيراد للحزم الأصلية المرخصة.
- تحديث فوري عند تشغيل التطبيق.
- تحديث يومي في 00:00 وفحص تأكيدي في 00:15 بتوقيت عمّان.
- فحص احتياطي كل ست ساعات.
- منع Release العام حتى اكتمال اللغات الثلاث والنصوص اليومية الأصلية.
- رفع الاختبارات إلى 55 اختبارًا.
