# ملاحظات الإصدار 4.1.1 — إصلاح تحديث اللغات اليومية

- يطلب التطبيق الآن مسار البيانات الموقّع الخاص باللغة المختارة: العربية أو اليونانية أو الإنجليزية.
- يحتفظ بمساري التقويم الموحّدين كنسخة احتياطية آمنة إذا لم تُنشر لغة معينة.
- يفحص التحديث عند كل عودة حقيقية للتطبيق إلى الواجهة باستخدام ETag خاص بكل رابط، بدل الفحص مرة واحدة فقط طوال بقاء عملية أندرويد.
- يمنع استخدام ETag أو ملف موقّع خاص بلغة مع لغة أخرى.
- يحدّث اللغة فور تبديلها من الإعدادات.
- يفحص WorkManager الخادم دوريًا حتى عندما تكون بيانات اليوم موجودة، بحيث تصل التصحيحات المنشورة في اليوم نفسه.
- تنظف GitHub Actions مسارات اليوم والـcurrent قبل توليد اللغات، فلا يمكن نشر ملف قديم للغة فشلت اليوم.
- أصبحت ملفات اللغات الثلاث متوافقة مع مخطط أندرويد الكامل وتحافظ على الخدمات والقراءات.
