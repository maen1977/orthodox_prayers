# ملاحظات إصدار Orthodox Prayers 5.0.0

## أهم التغييرات

- رفع التطبيق إلى Android API 36 وAndroid Gradle Plugin 8.13.2 وGradle 8.13.
- توحيد عقد بيانات الإصدار بين مولّد البيانات، الاختبارات، التطبيق، وGitHub Actions.
- فصل التخزين الموقّع لكل لغة: العربية والإنجليزية واليونانية.
- الاحتفاظ بآخر 30 يومًا موثوقًا لكل لغة مع نسخة حالية ونسخة احتياطية ذرية.
- التحقق من الخدمات السبع داخل الهاتف بعد التحقق من التوقيع وقبل العرض.
- حصر التنزيل في HTTPS والمضيف الرسمي ومنع التحويلات التلقائية.
- إضافة ثيم ورقي وليلي للقارئ، سطوع مستقل، نسبة تقدم، وملاحظات شخصية محلية.
- إضافة سجل بحث سريع مع إمكانية مسحه.
- إضافة ساعات هدوء وقنوات إشعارات منفصلة وفتح الإشعار في الصفحة المقصودة.
- إضافة Widget للشاشة الرئيسية يعرض التاريخ والعيد والصيام من البيانات الموقعة المحلية.
- إضافة Baseline/Startup Profiles لمسارات بدء التطبيق والقارئ والبحث.
- إضافة SBOM، تقرير اعتماديات، أرشيف مصدر نظيف، وبصمات SHA-256 تلقائية للإصدار.
- مزامنة بيانات 17 يوليو 2026 الموقعة مع أصول التطبيق.

## الخصوصية

لا إعلانات، لا حسابات، لا تتبع، ولا رفع للملاحظات أو سجل القراءة أو سجل البحث. تبقى هذه البيانات داخل الجهاز.

## غير مضاف عمدًا

لم تتم إضافة خاصية الصوت بناءً على طلب صاحب المشروع.
