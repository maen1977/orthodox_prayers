# Orthodox Prayers 4.1.5 — نسخة المصدر المنقحة

تطبيق أندرويد أرثوذكسي يساعد المؤمن في متابعة قراءات اليوم، الصيام، المناسبات، الصلوات والخدمات، والبحث في النصوص المتاحة دون اتصال بعد تنزيل التحديث الموقّع.

## المبدأ الديني غير القابل للتجاوز

المشروع يحتوي ثلاثة مسارات مستقلة تمامًا:

- **العربية:** نص عربي رسمي فقط، بالأولوية: مطرانية الروم الأرثوذكس في الأردن، ثم بطريركية القدس العربية، ثم بطريركية أنطاكية العربية.
- **الإنجليزية:** نص إنجليزي منشور أصلًا من جهة أرثوذكسية رسمية مسجلة.
- **اليونانية:** نص يوناني منشور أصلًا من جهة كنسية يونانية أو أرثوذكسية رسمية مسجلة.

لا يترجم التطبيق نصًا دينيًا من لغة إلى أخرى، ولا يستخدم الذكاء الاصطناعي لإعادة الصياغة، ولا يضيف التشكيل أو النبرات آليًا. إذا لم يوجد نص رسمي أصلي للغة المختارة، يظهر **غير متوفر** بدل نص بديل.

## حماية النصوص

- يبقى نص العرض كما أُدخل من نسخة المصدر، مع المحافظة على محارف Unicode والتشكيل والنبرات.
- لكل نص وبند بحث بصمة SHA-256.
- تبسيط التشكيل أو الهمزات يحدث داخل فهرس بحث مشتق فقط، ولا يُعرض للمستخدم.
- بيانات اليوم لا يقبلها التطبيق إلا بعد التحقق من توقيع RSA/SHA-256.
- عند فشل التحديث تبقى آخر نسخة موقعة وسليمة.

## GitHub Actions

يوجد Workflowان فقط:

1. **Update** — تحديث الساعة 00:00 وفحص النشر الساعة 00:15 بتوقيت `Asia/Amman`.
2. **Build** — بوابة الجودة، ثم استيراد أحدث بيانات منشورة موقعة والتحقق منها، واختبارات Android، وLint Debug، وبناء Debug APK. يبقى بناء APK/AAB الموقّع محميًا للإصدارات فقط، بينما CodeQL والمحاكي متوقفان مؤقتًا حتى يستقر البناء الأساسي.

التحديث اليومي يسمح بإظهار النص غير المتوفر بصورة آمنة. أما إصدار Production الموقّع فيُحجب بواسطة `scripts/validate_release_readiness.py` إلى أن تكتمل حزم الخدمات ومكتبات الكتاب المقدس الأصلية في اللغات الثلاث.

- حزمة الخدمات العربية: 601/601 مقطعًا.

- القراءات غير المتوفرة لا تُملأ بترجمة أو بنسخة من لغة أخرى.

## البدء

اقرأ بالترتيب:

1. `GITHUB_UPLOAD_AR.md`
2. `GITHUB_SECRETS_AR.md`
3. `SETUP_AR.md`
4. `NATIVE_CONTENT_IMPORT_AR.md`
5. `RELEASE_READINESS_AR.md`

فحص المشروع محليًا:

```bash
python -m pip install -r requirements-dev.txt
python scripts/run_quality_gate.py --strict-native-lanes
```

الخيار `--strict-native-lanes` أصبح فحصًا فعليًا: يرفض نسخ العربية إلى مسار آخر ويرفض النص المكتوب بأبجدية خاطئة. لا يدّعي اكتمال الإنجليزية أو اليونانية؛ اكتمال المحتوى شرط مستقل للإنتاج تتحقق منه `scripts/validate_release_readiness.py`.

التحقق من Gradle Wrapper:

```bash
python scripts/verify_gradle_wrapper.py
```

إنشاء ZIP نظيف قابل للرفع، دون `.git` أو الكاش أو مفاتيح خاصة:

```bash
python scripts/create_clean_source_archive.py ../OrthodoxPrayers-4.1.5-refined-source.zip
```

## حدود النسخة الحالية

- لقطة المصدر المضمّنة تبقى نسخة كاملة موقعة تصلح للرجوع دون إنترنت، وليست وعدًا بأنها بيانات تاريخ تشغيلك.
- مهمتا Debug وRelease تستوردان أحدث بيانات منشورة موقعة من فرع `verified-data` بعد نجاح بوابة المصدر، وتتحققان منها قبل بناء التطبيق.
- الإنجليزية واليونانية غير مكتملتين بعد؛ لا تُملآن بترجمة آلية أو نص من لغة أخرى.
- لا ترفع أي ملف مفتاح خاص أو Keystore أو كلمات مرور إلى GitHub.
