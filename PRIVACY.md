# سياسة الخصوصية

**التطبيق:** الأجندة والصلوات الأرثوذكسية  
**المقدّم:** معن حنونة للستلايت  
**التواصل:** +962 78 827 2988

لا يطلب التطبيق إنشاء حساب، ولا يعرض إعلانات، ولا يضم أدوات تحليلات أو تتبع في الإصدار الحالي.

يستخدم التطبيق اتصال الإنترنت فقط لتنزيل ملف بيانات اليوم وتوقيعه المنفصل عبر HTTPS. قد يسجل مزود الاستضافة معلومات اتصال تقنية معتادة، مثل عنوان IP ووقت الطلب، وفق سياسة ذلك المزود.

تُحفظ الإعدادات والمفضلة وموضع القراءة ونسخ البيانات الموقعة محليًا على الجهاز. لا يرسل التطبيق هذه البيانات إلى خادم خاص بالمشروع.

لا يطلب التطبيق صلاحيات للموقع أو جهات الاتصال أو الكاميرا أو الميكروفون أو الصور أو ملفات المستخدم. الصلاحيات الحالية هي الإنترنت وحالة الشبكة فقط.

لا توجد حسابات مستخدمين أو بيانات حساب محفوظة لدى المشروع. يمكن حذف البيانات المحلية بمسح بيانات التطبيق أو إلغاء تثبيته.
