# تقرير ترقية Orthodox Prayers إلى 5.0.0

## النتيجة

تم تحويل النسخة إلى قاعدة إصدار أكثر صرامة واعتمادية مع المحافظة على النصوص الموقعة وعدم إعادة كتابة التطبيق كاملًا بطريقة قد تكسر الميزات الحالية.

## الأمان والبيانات

1. أضيف `DataContract` ليكون عقد إصدارات Schema موحدًا وواضحًا.
2. أضيف `NetworkEndpointSecurity` لحصر التنزيل في `raw.githubusercontent.com` عبر HTTPS فقط.
3. عُطل اتباع Redirect تلقائيًا.
4. أصبح فحص اكتمال الخدمات جزءًا من تحقق الهاتف، وليس فحص الخادم فقط.
5. أصبح الكاش منفصلًا لكل لغة ويحتفظ بـ30 يومًا موثوقًا.
6. بقي حفظ JSON والتوقيع ذريًا مع `current` و`backup` وأرشيف مؤرخ.
7. وُحدت حالة الإصدار الجزئي الموثوق مع بوابات الجودة دون قبول نص غير موثق.

## تجربة المستخدم

- ثيمات قارئ مستقلة: النظام، ورقي، ليلي.
- تحكم في سطوع شاشة القارئ.
- إظهار تقدم القراءة كنسبة مئوية.
- ملاحظات خاصة لكل صلاة تحفظ محليًا.
- سجل آخر عمليات البحث.
- ساعات هدوء للتذكيرات.
- قنوات إشعارات مستقلة وDeep Links داخلية.
- Widget يومي قابل لتغيير الحجم.

## البنية والأداء

تم فصل مسؤوليات إضافية عن `DataRepository` إلى عقد بيانات، سياسة شبكة، ومخزن يومي مستقل. لم يتم فرض إعادة كتابة شاملة إلى Compose أو Room؛ لأن البيانات الدينية الموقعة يجب أن تبقى كبايتات أصلية غير معدلة، والمخزن الذري الحالي أنسب لهذا الجزء. يمكن لاحقًا استخدام Room لفهرس البحث والبيانات الشخصية فقط دون المساس بحزمة النص الموقعة.

أضيفت Baseline Profile وStartup Profile للمسارات الأكثر استخدامًا، كما بقي RecyclerView للقارئ الطويل لتجنب تحميل النص كاملًا كواجهات دفعة واحدة.

## خط الإصدار

الإصدار المحمي ينتج:

- AAB موقّع.
- تقرير الاعتماديات المحلولة.
- CycloneDX SBOM.

## الاختبارات المنفذة في بيئة التنقيح

- 93 اختبار Python ناجحًا.
- تحقق التوقيعات المنفصلة للبيانات المضمنة والمؤرخة.
- تحقق Schema والخدمات واللغات الثلاث.
- تحقق الأسرار وGradle Wrapper وملفات Workflows.
- تحقق جاهزية نصوص الرسالة والإنجيل بالعربية والإنجليزية واليونانية.

بناء Android المحلي لم يُنفذ في بيئة التنقيح لعدم وجود Android SDK وحزمة التوقيع الخاصة. GitHub Actions مهيأ لتنفيذ Unit Tests وLint وبناء Debug، ثم APK/AAB الموقّعين داخل بيئة `production` عند توفر الأسرار.
