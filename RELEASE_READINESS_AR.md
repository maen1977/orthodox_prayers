# حالة جاهزية Orthodox Prayers 5.0.16 R20

## الحكم الحالي

المشروع سليم تقنيًا وقابل للاختبار، لكنه **غير جاهز لإصدار Production ديني كامل**.

الاكتمال الكنسي المثبت من قائمة 15 خدمة إلزامية:

- العربية: 0/15.
- الإنجليزية: 0/15.
- اليونانية: 0/15.

أما امتلاء حقول JSON فهو العربية 539/539، والإنجليزية 774/774، واليونانية 766/766. هذه أرقام سلامة تقنية فقط، ولا تُستخدم لإعلان اكتمال خدمة كنسية.

## ما أصبح محميًا في R20

- لا ترجمة آلية ولا رجوع من لغة إلى أخرى.
- كل حزمة خدمات وكل مسار يومي مستقل بلغته.
- تُدرج قراءات اليوم داخل القداس عبر مواضع ثابتة في طبعات العربية والإنجليزية واليونانية.
- تزال بقايا USFM من العرض من دون تغيير كلمات المصدر.
- لا تحذف الأقواس المربعة الصحيحة من النص الكتابي على أنها علامات مؤقتة.
- لا يعرض دليل الكنائس اسمًا عربيًا داخل الواجهة الإنجليزية أو اليونانية عند غياب الاسم المترجم.
- لا تظهر عبارة «القداس الكامل» أو نسبة 100% بناءً على امتلاء الحقول.

## النقص الذي يمنع Production

- مكتب نصف الليل.
- الساعات الأولى والثالثة والسادسة والتاسعة.
- التيبيكا.
- التقدمة.
- قداس القديس باسيليوس.
- قداس القدسات السابق تقديسها.
- الغروب والسحر والنوم الصغرى الموجودة حاليًا مختصرة.
- اكتمال قداس القديس يوحنا الذهبي الفم غير مثبت بعد على طبعة واحدة كاملة ومصرح بها لكل لغة مع القطع المتغيرة.
- صلوات ما قبل المناولة جزئية بالعربية وغير متوفرة كنص كامل بالإنجليزية واليونانية.
- صلوات الشكر بعد المناولة غير متوفرة كنص كامل في اللغات الثلاث.

التصريح الحاكم موجود في:

```text
canonical/religious_completeness_manifest.json
```

## أوامر الفحص

بوابة سلامة المصدر:

```bash
python scripts/run_quality_gate.py --strict-native-lanes
```

فحص صدق قائمة الاكتمال:

```bash
python scripts/validate_religious_completeness.py
```

حاجز Production المقصود أن يفشل الآن:

```bash
python scripts/validate_religious_completeness.py --require-production-complete
```

لا يجوز رفع حالة أي خدمة إلى `complete_exact_native_edition` إلا بعد إدخال نصها الأصلي الكامل من طبعة مصرح بها، وربطه بمصدر وبصمة، والتحقق من جميع أقسامها الثابتة والمتغيرة.

## التوقيع والحقوق

- بيانات اليوم لا تُنشر إلا بتوقيع المفتاح الخاص المحفوظ في GitHub Environment المحمي.
- فان دايك العربية، World English Bible، والعهد الجديد اليوناني البطريركي 1904 مسجلة كمواد ملكية عامة.
- يجب الاحتفاظ بوثائق الإذن الخارجية لكل طبعة ليتورجية مقيدة قبل التوزيع العام.
- لا يحتوي المستودع على مفتاح توقيع خاص أو Android Keystore.
