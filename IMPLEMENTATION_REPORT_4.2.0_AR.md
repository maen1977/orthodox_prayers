# تقرير تنفيذ Orthodox Prayers 4.2.0

## ما تم تنفيذه

### تجربة الاستخدام

- تقويم شهري مع التنقل بين الأشهر، تمييز اليوم الحالي، وفتح الأيام التي تتوفر لها بيانات موثوقة.
- عرض غريغوري أو يولياني قديم من الإعدادات. التحويل البسيط الحالي مخصص للفترة 1900–2099 حيث الفرق 13 يومًا.
- سجل لآخر 20 نصًا مفتوحًا مع زر لمسح السجل.
- تثبيت النصوص وإظهار أول ثلاثة نصوص مثبتة في الصفحة الرئيسية.
- مجموعات للمفضلة: يومية، طقسية، شخصية، وغير مصنفة، مع تحريك العناصر للأعلى والأسفل.

### القراءة

- اختيار خط Sans أو Serif أو Monospace.
- تعديل تباعد الأسطر وحجم الخط.
- تمرير تلقائي بأربع سرعات، ويتوقف عند لمس المستخدم أو بلوغ نهاية النص.
- حفظ موضع القراءة، والوضع الليلي، ومنع انطفاء الشاشة.

### التذكيرات

- صلاة الصباح، قراءات اليوم، صلاة المساء، الأعياد والتذكارات، حالة الصيام، وتذكير شخصي.
- كل تذكير معطل افتراضيًا ويحتاج اختيار المستخدم.
- وقت كل تذكير قابل للتغيير على خطوات 30 دقيقة.
- لا يرسل تذكير العيد أو الصيام نصًا ناقصًا في اللغة المختارة.

### التحديث والأمان

- WorkManager 2.11.2.
- فحص احتياطي كل 12 ساعة إضافة إلى الفحص عند فتح التطبيق ومهمتي 00:00 و00:15 بتوقيت عمّان.
- بقي التحقق عبر HTTPS والتوقيع المنفصل ومخطط JSON والعودة إلى آخر نسخة سليمة.
- بقي مفتاح التوقيع خارج المستودع ولا يظهر إلا بعد انتهاء جلب المصادر وفحص المخرجات غير الموقعة.

### الإنجليزية واليونانية

تم فحص جميع حقول الحزمتين. كل منهما يحتوي حاليًا 132 من أصل 601 حقلًا. الحقول الناقصة وعددها 469 متركزة أساسًا في القداس الإلهي الكامل.

أضيفت الأدوات التالية:

```bash
python scripts/export_missing_native_fields.py --language en --output docs/native_missing_en.json
python scripts/export_missing_native_fields.py --language el --output docs/native_missing_el.json
python scripts/merge_authorized_native_services.py --input /secure/authorized-en.json
python scripts/merge_authorized_native_services.py --input /secure/authorized-el.json
```

أداة الدمج لا تترجم ولا تعيد الصياغة. تتحقق من اللغة، ومعرف المصدر، ودليل الإذن، ولا تملأ إلا الحقول الفارغة. لم تُنسخ نصوص كاملة من صفحات الإنترنت لأن المشروع يشترط نسخة أصلية مصرّحًا بها يمكن توثيقها وحفظ بصمتها.
