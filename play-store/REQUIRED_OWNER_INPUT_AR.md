# معلومات وإجراءات يكمّلها مالك المشروع

1. بريد دعم عام فعلي.
2. تفعيل GitHub Pages من `main` والمجلد `/docs` وفحص رابط الخصوصية.
3. لقطات شاشة حقيقية من APK/AAB المجرب.
4. إجابات Play Console النهائية: Data safety، الجمهور، Content rating، والحقوق.
5. استيراد حزم النصوص الرسمية المأذون بها لكل لغة وفق `NATIVE_CONTENT_IMPORT_AR.md`.
6. حفظ مفاتيح Android وتوقيع البيانات خارج المستودع، وإضافتها إلى GitHub Secrets فقط.
7. الاختبار الداخلي أو المغلق الذي يطلبه حساب Play Console.

لا ترفع أي ملف JKS أو PEM خاص أو كلمات مرور أو مستندات موافقات خاصة إلى المستودع العام.
